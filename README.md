# React_Blog
CRUD Blog using React, NodeJS, Experss  

### Installing

- Install the packages mentioned in package.json file for getting all dependencies of the project.
  ```
  npm install --save
  ```
- Starting the application
  <br/>
  *To start both react and api*
  ```
  npm run dev
  ```
  *Only for api*
  ```
  npm start
  ```
